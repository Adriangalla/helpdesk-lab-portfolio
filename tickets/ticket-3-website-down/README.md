# Company Website Not Accessible

## Issue Summary
User reported that the company website was not accessible.

## Troubleshooting Steps
- Attempted to access website locally — unsuccessful
- Performed DNS lookup — successful
- Tested connectivity — no response
- Attempted access from external network — unsuccessful

## Findings
- DNS working
- Server not responding
- Likely server-side issue

## Resolution
Escalated to appropriate team due to confirmed server-side issue impacting multiple users.


## Skills Demonstrated
- DNS troubleshooting
- Network connectivity testing
- Issue isolation
- Escalation procedures
